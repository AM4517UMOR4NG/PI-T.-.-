A. AUTENTIKASI & AKSES
Use-Case Name Login
Use Case ID RPL-UC001
Author Alogo Situmorang
Priority High
Primary Actor Semua Pengguna (Pelanggan, Admin, Kasir, Pemilik)
Description Proses autentikasi untuk masuk ke sistem menggunakan email dan password.
Precondition Pengguna sudah memiliki akun terdaftar.
Typical Course Actor Action System Response
1. Klik tombol "Login"
2. Tampilkan form login
3. Masukkan email & password
4. Validasi kredensial
5. Buat session
6. Redirect ke dashboard sesuai role
Postcondition User berhasil login dan dapat mengakses fitur sesuai role.
